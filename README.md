# create-mastodon-component

## Install
```bash
npm i -g create-mastodon-component
```

## Example
Run the following command from the root directory of **mastodon**.
```bash
create-react-component table
```
Creates:
```
Table
├── Table.js
├── style.scss
└── index.js
└── __tests__
        └── TableTest.js
```
